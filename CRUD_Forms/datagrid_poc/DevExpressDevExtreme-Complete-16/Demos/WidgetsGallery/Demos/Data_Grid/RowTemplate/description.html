The DevExtreme HTML/JS Data Grid (dxDataGrid) widget supports custom row templates. This feature provides several methods for defining a template markup. In this demo, the Underscore template engine is used. A custom row template is factored out into a separate element with a specified name (see the Markup demo tab). To set this template for a grid row, the template name is assigned to the grid's rowTemplate configuration option. To enforce using the Underscore template engine, the DevExpress.ui.setTemplateEngine function is called with 'underscore' passed as a parameter.
